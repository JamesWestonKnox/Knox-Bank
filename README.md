Install:

1. Install Node.js
2. Clone Git repo
3. Open repo in VSCode.
4. Open terminal for project
5. navigate to backend folder using: cd backend.
6. run: npm install.
7. navigate to frontend folder using: cd frontend.
8. run: npm install.
9. Dependancies sorted now.
10. to run and test frontend and backend - open a terminal in each folder and run: npm run dev
