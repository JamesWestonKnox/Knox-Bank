# Knox Bank
Knox bank is an international online banking system, allowing customers to send international payments and employees to verify payments with swift codes.

# Features
- **Customer Portal**:

    Customers can login to our secure customer portal, where they can enter payee account details and send secure swift        transactions internationally.
- **Employee Portal**:

    Knox Bank employees can login to our employee portal using their pre registered employee accounts. Here they can view all pending transactions and verify them to be sent off to swift.

# Installation
1. Install NODE.js
2. Clone github repository.
3. Open repository in VSCode.
4. Open terminal in root of repository.
5. Navigate to backend folder using: cd backend.
6. Run: npm install.
7. Navigate to frontend folder using: cd frontend.
8. Run: npm install.
9. To run and test frontend and backend - open a terminal in each folder and run: npm run dev

# Using the application
## Customer
- If registered, enter login information and proceed to portal.
- If new account, click on register link and create an account.
- Once in portal, you may enter transaction details and click pay.
## Employee
- Click on button to take you to employee login page.
- Enter pre-registered employee account details.
- Once in employee portal, all unverified transactions are in a table on the left, click verify on them to send them to swift.
